# TO-DO App

### Dependências
``` 
Ruby 3.2.2
Rails 7.0.5
Node 18.13.0
```
---
### Instalação
```Bash
git clone https://github.com/jvlp/todo-app
cd todo-app
bundle install
cd fronend
yarn # ou npm -i
```
---
### Execução
#### API
```Bash
# Na pasta todo-app
rails s
```
#### Frontend
```Bash
# Na pasta todo-app/frontend
yarn dev
```
